Dynamic website built on Google's Cloud Platform

I was proposed a theoretical business problem where a company wanted to have a ski resort website for users to book vacations. This was my first ever dynamic website built. I built the functionality of the website and deployed it through Google's cloud service.


Technologies Used

Website functionalities include end-user logins/registering, displaying a list of available ski resort packages, and allowing users to pay.
