Dynamic website built on Amazon's Cloud Platform

I was proposed a theoretical business problem where a company wanted to have a cloud-based video game platform. There was a higher emphasis on security, so no frontend was built to compensate.I was the backend web developer who was responsible for building the functionality of the website, securing it, and deploying it through Amazon's cloud service.


Technologies Used

Website functionalities include end-user logins/registering, profile editing, displaying a list of games from a database, allowing users to play games, allowing developers to securely edit and add games to the website, and having payment functionality. This website was built with security as its top priority. The environment was built using a dedicated development laptop and used ASP.NET MVC.
