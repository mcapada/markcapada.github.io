SKILLS ONTARIO - MAY 2019

The scope for the competition included some areas that weren't even covered in post-secondary studies. The three main areas in the provincial competition were Linux Server, Windows Server, and Routing and Switching. This included Windows Server administration involving ADDS, DHCP, and DNS. Cisco routers and switches were configured using various protocols (including OSPF and VLAN) and network addresses had to be planned and allocated correctly. Finally, Ubuntu Server CLI administration involved FTP, SSH, Apache2 and Samba.

SKILLS CANADA @ NOVA SCOTIA - 2019

The National competition was similar to Provincials, but had a much wider scope. In addition to the tasks in Windows Server, implementations of RDP, GPO, and network monitoring had to be performed. LDAP, DHCP, security management and RAID partitioning were expected of competitors in the Linux Server stage. The switched network provided was large scale and involved protocols such as NAT, Ether-Channel, ACLs, VPN Tunnels and Port Security in order to work.
