Information Technology for Me
(IT4ME)

  • Webapp Demo YouTube
  • Source Code Github
  • Date Sept 2019 - Apr 2020

For our Degree project, we decided to create a web application that displayed Information Technology post-secondary education programs to end-users with hopes to branch out to different fields in the future. This also included building a fully-functional physical network under a controlled domain. In my group of six, I was the backend web developer who was responsible for building the functionality of the website, securing it, and deploying it through Microsoft Azure's cloud service.

Website functionalities include end-user logins/registering, profile editing, displaying program information from a database, showing locations on a map, and having payment functionality. This website was built with security in mind. The environment was built using a dedicated development laptop and used ASP.NET MVC.
